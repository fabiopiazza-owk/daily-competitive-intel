### 🥇 Disease-context-conditioned generative discovery layer
**Rationale:** DrugGen-2 directly threatens K's differentiation in target-to-lead. We need disease-context conditioning (indication + progression stage + patient stratum) as a first-class parameter across our discovery workflows before customers ask why our competitor has it and we don't.
**Effort:** 6–8 weeks for MVP conditioning API on top of existing generative modules; 1 quarter for full multimodal integration.

### 🥈 Regulatory-grade LLM causality assessment module for pharmacovigilance
**Rationale:** Peer-reviewed evidence (WHO-UMC study) that LLMs can hit causality-assessment benchmarks. High-value, differentiated feature for enterprise pharma safety teams; strong lock-in via signal-detection workflows. Also anticipates HHS's shifting posture on deprescribing evidence.
**Effort:** 8–10 weeks, leveraging existing RAG infrastructure + FAERS integration.

### 🥉 Long-horizon agent memory + dense-reward eval harness
**Rationale:** Trial design, protocol iteration, and biomarker discovery are all long-horizon tasks where K's agents currently degrade. Proactive Memory Agent + Long-Horizon-Terminal-Bench give us both the primitive and the eval methodology to leapfrog Biomni on autonomous R&D workflows.
**Effort:** 4 weeks for eval harness; 6–8 weeks for memory primitive integrated into agent runtime.
