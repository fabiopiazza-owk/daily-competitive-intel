## 6. Build Signals for K Platform

### Rank 1: **Spatial Transcriptomics Temporal Modeling Module**
**Rationale:** stVCR (Nature Methods) demonstrates that modeling spatiotemporal single-cell dynamics from spatial transcriptomics is now feasible and high-impact. K Platform already handles spatial pathology — adding temporal dynamics modeling would differentiate our biomarker discovery for tumor evolution and treatment response prediction. This directly strengthens the AstraZeneca/Sanofi value proposition for longitudinal trial analysis.

### Rank 2: **Pharmacovigilance / Adverse Event Signal Detection Module**
**Rationale:** The FDA's new unified AE reporting system creates a structured data source we can leverage. Building an AI-powered safety signal detection module positions K Platform as a post-market analytics tool — expanding TAM beyond discovery and trials into lifecycle management. Pharma enterprise buyers increasingly want end-to-end platforms. Scoping effort: integrate RAG over FAERS data + LLM summarization with GxP audit trails.

### Rank 3: **GxP-Safe Agentic Query Layer with Instruction Hierarchy**
**Rationale:** OpenAI's prompt injection defenses and instruction hierarchy work, combined with their new agent runtime, signal that agentic AI is maturing fast. K Platform should build a GxP-compliant agentic interface that lets pharma scientists query multimodal data (pathology + genomics + clinical) conversationally, with full auditability and prompt injection resistance. This is a defensive play: if we don't build it, Benchling or Biomni will layer it on top of commodity models and erode our UX moat.
